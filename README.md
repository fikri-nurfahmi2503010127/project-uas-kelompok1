Project UAS Kelompok 1

Tema : Sistem Akademik Sederhana

Program berbasis C++ (Console/Terminal) untuk mengelola data siswa, mata pelajaran, jadwal pelajaran, nilai siswa, pencarian data, dan pengurutan data akademik menggunakan struktur data yang telah dipelajari pada mata kuliah Algoritma dan Struktur Data.
